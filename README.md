# RandomUserAPI
## _Criar uma **API** que retorna informações de um usuário aleatorio_ ❓🤔

##### DESAFIO 💯

_As informações dos usuários devem conter, pelo menos:_

- Primeiro nome
- Segundo nome
- Gênero
- Data de nascimento
- Email
- Senha

**Ferramenta** 🔧
- Node.js
